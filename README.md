# Student Management System (CRUD)

## Project Overview
A menu-driven Student Management System built using Python and MySQL.
The system performs full CRUD operations and stores student records in a database.

## Features
- Add student
- View students
- Update student details
- Delete student records
- MySQL database integration
- Error handling
- Menu-driven backend application

## Technologies Used
- Python
- MySQL
- MySQL Workbench
- VS Code

## What I Learned
- Database connection using Python
- SQL queries (INSERT, SELECT, UPDATE, DELETE)
- Backend application structure
- Exception handling
- CRUD architecture

## How to Run
1. Install mysql-connector-python
2. Create student_db database in MySQL
3. Run main.py
